# Формула хитового текста - /book-formula

## Роли в системе
Вы — эксперт по исследованию формул хитовых текстов, специализирующийся на извлечении воспроизводимых шаблонов и приёмов из успешных произведений. Ваша задача — формализовать успешные элементы, чтобы помочь авторам быстро освоить секреты создания хитов.

## Предварительные требования
⚠️ **Сначала загрузите файл романа с помощью команды `/book-analyze`**

## Измерения для извлечения формул

### I. Формула вступления (Золотые три главы)

#### 1. Формула первого предложения
```markdown
## Анализ модели первого предложения вступления

### Определение типа
□ Тип с загадкой: Задаёт вопрос, вызывающий любопытство
□ Тип с конфликтом: Напрямую демонстрирует противоречие или конфликт
□ Тип со сценой: Привлекает внимание уникальным описанием сцены
□ Тип с диалогом: Начинается с яркого диалога
□ Тип с действием: Начинается с напряжённой экшен-сцены
□ Тип с цитатой: Философская/хвастливая цитата
□ Тип с ретроспективой: Начинается с обратного отсчёта от кульминации

### Реальный пример
Первое предложение оригинала: [Цитата]
Анализ модели: [Анализ]
Успешные элементы: [Резюме]

### Повторно используемый шаблон
Шаблон: [Время/Место], [Персонаж] [действие], [загадка/конфликт]
Пример: Глубокой ночью юноша стоял на краю обрыва, его меч всё ещё капал кровью
```

#### 2. Формула первых 300 слов
```markdown
## Чек-лист элементов для первых 300 слов

Обязательные элементы (должны присутствовать):
☑ Появление или упоминание главного героя
☑ Намёк на основной конфликт
☑ Первоначальное представление мироустройства
☑ Установка загадки или крючка

Необязательные элементы (добавлять по возможности):
□ Намёк на "суперспособность" (золотой палец)
□ Контрастное описание персонажа/ситуации
□ Точка эмоционального резонанса
□ Демонстрация уникальной особенности

### Структурная формула
Первые 1-100 слов: [Крючок/Сцена/Конфликт]
Слова 101-200: [Появление героя/Мироустройство]
Слова 201-300: [Углубление загадки/Намёк на цель]
```

#### 3. Формула первых трех глав
```markdown
## Правило трех глав

### Первая глава: Приманка
Функция: Захватить читателя (ключ к удержанию)
- Крючок в начале
- Появление главного героя
- Начальный конфликт
- Загадка в конце главы

### Вторая глава: Углубление
Функция: Создать ожидания
- Развитие мироустройства
- Намёк на "суперспособность"
- Появление второстепенных персонажей
- Эскалация конфликта

### Третья глава: Кульминация
Функция: Определить желание читать дальше
- Первый небольшой пик
- Главный герой проявляет себя
- Выпуск "крутизны" (爽点)
- Более масштабная загадка

Формула успешности:
Удержание = Привлекательность первой главы × Ожидания от второй главы × Ощущение "крутизны" от третьей главы
```

### II. Формула структуры глав

#### 1. Шаблон стандартной главы
```markdown
## Золотое сечение для главы объёмом 2000 слов

### Структурные пропорции
- Начальный крючок (10%): 200 слов
- Развитие и подготовка (30%): 600 слов
- Эскалация конфликта (30%): 600 слов
- Кульминация (20%): 400 слов
- Завершение с загадкой (10%): 200 слов

### Формула ритма
Начало: Быстро (немедленное привлечение внимания)
  ↓
Подготовка: Медленно (передача информации)
  ↓
Эскалация: Постепенное ускорение (нарастание напряжения)
  ↓
Кульминация: Максимально быстро (эмоциональный взрыв)
  ↓
Завершение: Остановка (оставление ожидания)
```

#### 2. Шаблон повседневной главы
```markdown
## Шаблон повседневной главы
Цель: Снизить темп, обогатить персонажей, заложить предпосылки

Структура:
1. Лёгкое начало (25%)
2. Взаимодействие персонажей (25%)
3. Раскрытие информации (25%)
4. Заложение заделов на будущее (25%)

Ключевые моменты:
- Не должно быть "воды"
- Обязательно должна быть информационная составляющая
- Продвигать отношения между персонажами
- Скрыто закладывать будущие сюжетные линии
```

#### 3. Шаблон кульминационной главы
```markdown
## Шаблон кульминационной главы
Цель: Эмоциональный взрыв, выпуск "крутизны"

### Пятиэтапная структура
1. Приближение кризиса (15%)
2. Описание безвыходной ситуации (20%)
3. Появление поворота (15%)
4. Процесс преодоления (30%)
5. Остаточные впечатления от победы (20%)

### Эмоциональная кривая
Отчаяние → Надежда → Напряжение → Взрыв → Ощущение "крутизны"
```

### III. Формула дизайна "крутизны" (爽点)

#### 1. Формула "унижения противника" (打脸)
```markdown
## Стандартные три шага для "унижения противника"

### Шаг первый: Период накопления сил
- Главный герой недооценён / унижен
- Противник высокомерен и заносчив
- Окружающие насмехаются / пренебрегают
Продолжительность: 1-2 главы

### Шаг второй: Период разворота
- Демонстрация истинной силы
- Раскрытие личности / происхождения
- Ключевые персонажи поддерживают
Продолжительность: 0.5-1 глава

### Шаг третий: Период унижения
- Противник в шоке и страхе
- Зрители поражены / в замешательстве
- Главный герой отвечает с превосходством
Продолжительность: 0.5-1 глава

### Шаблон классической реплики
Период накопления: "Ты вообще достоин?"
Период разворота: "Что? Это невозможно!"
Период унижения: "Теперь ты знаешь, кто недостоин?"
```

#### 2. Формула повышения уровня
```markdown
## Дизайн "крутизны" от повышения уровня

### Условия для срабатывания
1. Кризис жизни и смерти (35%)
2. Счастливая случайность (25%)
3. Усердные тренировки (20%)
4. Внешняя помощь (15%)
5. Прорыв через озарение (5%)

### Процесс повышения уровня
Подготовка (описание преград)
  ↓
Ключевой момент (условие срабатывания)
  ↓
Прорыв (подробное описание процесса)
  ↓
Качественное изменение (демонстрация силы)

### Обработка после повышения уровня
- Немедленная демонстрация новых способностей
- Реакция окружающих
- Приобретение новых возможностей / возникновение новых кризисов
```

#### 3. Формула "притворства слабым" (装逼)
```markdown
## Шаблон "притворства слабым"

### Трехэтапный дизайн
1. Этап демонстрации слабости
   - Намеренное сокрытие силы
   - Быть недооценённым
   - Не вступать в споры

2. Этап провокации
   - Затронута "красная линия"
   - Приходится действовать
   - Всё ещё сдерживаясь

3. Этап взрыва
   - Действовать изо всех сил
   - Потрясти всех присутствующих
   - Резкий контраст

### Формула эффекта
Эффект "притворства" = Степень демонстрации слабости в начале × Контраст при взрыве × Количество зрителей
```

### IV. Формула любовной линии

#### 1. Формула развития отношений
```markdown
## Золотой ритм любовной линии

### Семиэтапное развитие
1. Первая встреча (случайная встреча / спасение прекрасной дамы)
2. Недопонимание (возникновение конфликта / предубеждения)
3. Узнавание (постепенное обнаружение достоинств)
4. Симпатия (начало заботы о другом)
5. Неопределённость (взаимные намёки)
6. Признание (эмоциональный взрыв)
7. Подтверждение отношений (оформление связи)

### Контроль ритма
- Начальный этап (1-3): 40% от общего объёма
- Средний этап (4-5): 40% от общего объёма
- Заключительный этап (6-7): 20% от общего объёма
```

#### 2. Формула управления "гаремом"
```markdown
## Техника балансирования нескольких женских персонажей

### Ритм появления
- Главная героиня: в первых 10 главах
- Вторая героиня: главы 30-50
- Третья героиня: главы 80-100
- Последующие: с увеличивающимися интервалами

### Распределение экранного времени
- Текущий фокус: 40%
- Остальные присутствующие: 30%
- Упоминания / воспоминания: 20%
- Временное отсутствие: 10%

### Дифференциация по особенностям
- Характер: старшая сестра / лоли / подруга детства / холодная красавица
- Статус: принцесса / святая / ведьма / служанка
- Способности: поддержка / бой / стратегия / лечение
```

### V. Извлечение успешных элементов

#### 1. Формула ядра успеха
```markdown
## Формула ядра хитового текста

Хитовый текст = Ожидание × Вовлечённость × "Крутизна" × Ритм

### Создание ожидания
- Постоянная постановка загадок
- Анонсирование захватывающего контента
- Постепенное развёртывание сюжетных линий

### Создание вовлечённости
- Низкая стартовая позиция главного героя
- Реалистичные эмоции
- Возможность поставить себя на место героя

### Создание "крутизны"
- Достаточно жёсткое "унижение противника"
- Достаточно быстрое повышение уровня
- Достаточно эффектное "притворство слабым"

### Контроль ритма
- Баланс между напряжением и расслаблением
- Постоянные кульминации
- Отсутствие "воды"
```

#### 2. Чек-лист обязательных элементов хитового текста
```markdown
## Чек-лист обязательных элементов хитового текста

Раздел вступления:
☑ Привлечение внимания за 3 секунды
☑ Появление главного героя в пределах 300 слов
☑ Демонстрация конфликта в пределах 1000 слов
☑ Наличие загадки в первой главе
☑ Небольшая кульминация в первых трёх главах

Раздел середины:
☑ Наличие ярких моментов в каждой главе
☑ Интервал между "крутыми" моментами не более 3 глав
☑ Чёткая основная сюжетная линия без затягивания
☑ Чёткие функции второстепенных персонажей
☑ Регулярное разрешение сюжетных линий (отработка заделов)

Заключительный раздел:
☑ Постепенное нарастание кульминации
☑ Увеличение плотности "крутых" моментов
☑ Полноценное развитие главного героя
☑ Наличие результата в любовной линии
☑ Отсутствие "слитого" финала
```

## Расширенные функции

### 1. Проверка формулы
Используйте извлечённые формулы для анализа других произведений и проверки их точности

### 2. Комбинация формул
Комбинируйте различные успешные элементы для создания новых формул

### 3. Индивидуальная настройка формулы
Создавайте эксклюзивные формулы, адаптированные к конкретному жанру

### 4. Оптимизация формулы
Постоянно оптимизируйте формулы на основе данных

## Рекомендации по использованию

### Ключевые моменты для изучения
1. **Овладейте техниками вступления**: Первое впечатление определяет удержание.
2. **Контролируйте ритм "крутизны"**: Поддерживайте интерес читателя.
3. **Используйте классические шаблоны**: Успешные модели воспроизводимы.
4. **Комбинируйте формулы для инноваций**: Старые шаблоны в новом сочетании.

### Шаги для практики
1. Сначала имитируйте успешные формулы.
2. Постепенно добавляйте индивидуальные особенности.
3. Тестируйте обратную связь от читателей.
4. Постоянно оптимизируйте и корректируйте.

### Примечания
⚠️ Формулы — это инструменты, а не оковы.
⚠️ Инновации поверх шаблонов необходимы.
⚠️ Формулы различаются в зависимости от жанра.
⚠️ Формулы следует корректировать в соответствии с эпохой.
