# Усиление стиля Ван Юя

Преобразуйте или усильте существующий текст в стиле Ван Юя, сохраняя логику истории и добавляя характерные черты Ван Юя.

## Как использовать

```
/wangyu-enhance [текст или путь к файлу] [опции]
```

## Режимы усиления

### Легкое усиление (--light)
- Корректировка повествовательного тона
- Стандартизация терминологии системы силы
- Добавление небольшого количества описаний окружения

### Среднее усиление (--medium) по умолчанию
- Переписывание диалогов и действий
- Увеличение описания дефицита ресурсов
- Вставка описания психологических переживаний персонажа
- Умеренное добавление саспенса

### Глубокое усиление (--heavy)
- Полная реконструкция абзацев
- Добавление деталей мироустройства
- Переработка характера персонажа
- Создание заделов на будущее и саспенса
- Корректировка темпа сюжета

## Примеры усиления

### Оригинал
```
Чжан Сан вошел в пещеру, увидел много сокровищ и радостно их собрал.
```

### Легкое усиление
```
Чжан Сан осторожно ступил в пещеру, его духовное чутье просканировало пространство, обнаружив несколько магических артефактов, сваленных в углу. Подавив радость в сердце, он сначала установил предупреждающие формации, прежде чем начать их собирать.
```

### Среднее усиление
```
Чжан Сан на мгновение задержался у входа в пещеру, убедившись в отсутствии активированных запретов, прежде чем медленно войти внутрь.

При свете ночного жемчуга он ясно видел обстановку внутри — три магических артефакта неплохого качества спокойно лежали на каменном пьедестале, рядом находилось несколько мешков для хранения.

«Пещера предшественника?» — в сердце Чжан Сана мелькнула мысль, но многолетний опыт заставлял его сохранять бдительность. Бесплатный сыр бывает только в мышеловке, эти вещи появились слишком легко.

Сначала он проверил магической силой, убедившись в отсутствии скрытых запретов, и только потом осторожно начал собирать. Даже так, он взял только половину, оставив другую половину как запасной путь.
```

### Глубокое усиление
```
У входа в пещеру Чжан Сан остановился.

Воздух был наполнен слабым ароматом трав, смешанным с едва уловимым запахом крови. Этот запах был ему хорошо знаком — здесь погиб какой-то культиватор, не более трех месяцев назад.

Он достал талисман обнаружения, золотой свет вспыхнул, и талисман превратился в десятки световых точек, вошедших в пещеру. Через мгновение точки вернулись, принеся информацию, которая заставила его нахмуриться.

В глубине пещеры действительно находилось немало духовных сокровищ: три магических артефакта, семь мешков для хранения и один полный котел для алхимии. Однако самая глубокая часть пещеры была областью, куда талисман обнаружения не мог проникнуть.

«Ложная тревога или есть другая тайна?»

Чжан Сан задумался на мгновение и достал из мешка для хранения труп-марионетку. Это было средство разведки, которое он купил за большие деньги и которое могло спасти ему жизнь в критический момент.

Труп-марионетка неуклюже вошел в пещеру, а сам он стоял в ста шагах, управляя им с помощью духовного чутья.

Через полчаса труп-марионетка благополучно вернулся с двумя мешками для хранения. Чжан Сан осмотрел их и обнаружил, что внутри были только низкоуровневые пилюли и чуть более тысячи низкосортных духовных камней.

«Приманка», — сделал он вывод.

Настоящие ценности, вероятно, находились в той области, куда нельзя было проникнуть. Но уровень опасности там, вероятно, намного превосходил его возможности.

Чжан Сан решительно повернулся и ушел. Проведя столько лет в мире культивации, он прекрасно понимал одну истину: жадность — основная причина гибели культиваторов.
```

## Ключевые моменты усиления

### 1. Стиль повествования
- Убрать восклицательные знаки, использовать точки.
- Уменьшить количество прилагательных, увеличить конкретные описания.
- Сохранять объективность от третьего лица.

### 2. Дополнение деталей
- Добавить элементы культивации, такие как магическая сила, духовное чутье.
- Дополнить конкретной информацией о времени, расстоянии.
- Увеличить описания атмосферы окружения.

### 3. Поведение персонажа
- Отразить осторожность и расчетливость.
- Продемонстрировать опыт и методы.
- Подчеркнуть взвешивание интересов.

### 4. Углубление сюжета
- Усложнить простые сюжеты.
- Добавить неопределенные факторы.
- Заложить потенциальные опасности.

## Опции конфигурации

- `--mode <light|medium|heavy>` - степень усиления
- `--keep-plot` - сохранить исходный сюжет без изменений
- `--add-suspense` - добавить саспенс
- `--focus <aspect>` - сфокусироваться на определенном аспекте

## Примечания

- Усиление не изменит основной сюжет.
- Ключевая информация из оригинала будет сохранена.
- Объем текста может быть умеренно увеличен.
- Рекомендуется сначала использовать `/wangyu-analyze` для анализа.

## Связанные команды

- `/wangyu-style` - активировать стиль Ван Юя
- `/wangyu-write` - прямое создание
- `/wangyu-analyze` - анализ стиля
