Генерация конкретного, выполнимого списка задач на основе плана творчества.

## Цель

Преобразование макроплана в микрозадачи, делая творческий процесс управляемым и отслеживаемым.

## Шаги выполнения

### 1. Загрузка плана

Запустите `{SCRIPT}` для загрузки:
- План творчества: `stories/*/creative-plan.md`
- Информация об архитектуре глав
- Временная шкала и зависимости

### 2. Генерация списка задач

Создайте `stories/*/tasks.md`, содержащий:

#### Основные задачи по написанию
```markdown
## Задачи по написанию

### Высокий приоритет [должно быть выполнено в первую очередь]
- [ ] [P0] Глава 1: Завязка (3000 слов)
- [ ] [P0] Профиль персонажа: Детальная проработка главного героя
- [ ] [P0] Мироустройство: Базовые правила

### Средний приоритет [нормальное продвижение]
- [ ] [P1] Главы 2-5: Развитие сюжета (3000 слов на главу)
- [ ] [P1] Профиль персонажа: Основные второстепенные герои
- [ ] [P1] Описание сцен: Важные локации

### Низкий приоритет [опциональное улучшение]
- [ ] [P2] Бонусная история: Предыстория персонажа
- [ ] [P2] Сборник материалов: Детальное мироустройство
```

#### Пояснения к маркировке задач
- `[P]` - может выполняться параллельно
- `[Зависимость:X]` - требует предварительного выполнения задачи X
- `[P0/P1/P2]` - маркер приоритета

### 3. Сортировка и группировка задач

- Группировка по приоритетам
- Определение зависимостей
- Маркировка параллельных задач
- Оценка времени выполнения

### 4. Генерация плана выполнения

```markdown
## План выполнения

### Первый этап (Неделя 1)
Параллельные задачи группы 1:
- Проработка главного героя [P]
- Базовые правила мироустройства [P]
- Черновик Главы 1 [P]

### Второй этап (Недели 2-3)
Последовательные задачи:
- Глава 2 [Зависимость:Глава 1]
- Глава 3 [Зависимость:Глава 2]

Параллельные задачи группы 2:
- Проработка второстепенных героев [P]
- Дизайн локаций [P]
```

### 5. Вывод статистики задач

- Общее количество задач
- Общий предполагаемый объем (в словах)
- Предполагаемое время завершения
- Ключевые этапы

Подсказка для следующего шага: начните выполнение `/write` или просмотрите детали конкретной задачи.
