Создание конкретного, исполнимого списка задач на основе плана творчества.

## Цель

Преобразование макроплана в микрозадачи, делая творческий процесс управляемым и отслеживаемым.

## Шаги выполнения

### 1. Загрузка документов плана

Запустите `{SCRIPT}` для загрузки:
- План творчества: `stories/*/creative-plan.md`
- Информация о структуре глав
- Временная шкала и зависимости

### 2. Создание списка задач

Создайте `stories/*/tasks.md`, содержащий:

#### Основные задачи по написанию

**Важно**: На основе пятой главы specification.md и распределения сюжетных линий в creative-plan.md, отметьте сюжетные линии, задействованные в каждой задаче по написанию.

```markdown
## Задачи по написанию

### Высокий приоритет [должно быть выполнено в первую очередь]

- [ ] [P0] **T001** - Глава 1: [Заголовок главы] (Целевое количество слов)
  - **Основная задача**: [Основная задача этой главы]
  - **Ключевые моменты сюжета**: [Конкретные сюжетные точки]
  - **Задействованные сюжетные линии**:
    - PL-XX([Название сюжетной линии]) ⭐⭐⭐ Основное продвижение
    - PL-YY([Название сюжетной линии]) ⭐ Фоновое сопровождение
  - **Точки пересечения**: [Если применимо] X-001([Описание точки пересечения])
  - **Зарождение/раскрытие завязки**: [Если применимо] Зарождение F-001 / Раскрытие F-002
  - **Должно включать**: [Список ключевых элементов]
  - **Крючок в конце главы**: [Установка саспенса]
  - **Зависимости**: Нет
  - **Выход**: `content/volume1/chapter-001.md`

- [ ] [P0] **T002** - Профиль персонажа: Детальная проработка главного героя
  - **Основная задача**: Доработка проработки главного героя
  - **Должно включать**: Характер, предыстория, способности, желания, страхи, дуга развития
  - **Зависимости**: Нет
  - **Выход**: `characters/protagonist.md`

### Средний приоритет [нормальное продвижение]

- [ ] [P1] **T005** - Глава 5: [Заголовок главы] (Целевое количество слов)
  - **Основная задача**: [Основная задача этой главы]
  - **Ключевые моменты сюжета**: [Конкретные сюжетные точки]
  - **Задействованные сюжетные линии**:
    - PL-01([Название сюжетной линии]) ⭐⭐⭐ Основное продвижение
    - PL-02([Название сюжетной линии]) ⭐⭐ Вспомогательное
  - **Точки пересечения**: Нет
  - **Зарождение/раскрытие завязки**: Зарождение F-001([Содержание завязки])
  - **Должно включать**: [Ключевые элементы]
  - **Крючок в конце главы**: [Саспенс]
  - **Зависимости**: T004 (Глава 4)
  - **Выход**: `content/volume1/chapter-005.md`

### Низкий приоритет [необязательное доведение до совершенства]

- [ ] [P2] Дополнительная глава: Предыстория персонажа
- [ ] [P2] Сборник описаний: Детальное мировоззрение
```

**Пояснения к полям задачи**:
- **Задействованные сюжетные линии**: Читать из столбца "Активные сюжетные линии" в creative-plan.md, отмечая, какие сюжетные линии продвигаются в этой главе.
- **Точки пересечения**: Читать из раздела 5.3 specification.md, отмечая, является ли эта глава точкой пересечения.
- **Зарождение/раскрытие завязки**: Читать из раздела 5.4 specification.md, отмечая операции с завязками, задействованные в этой главе.

#### Пояснение к маркировке задач
- `[P]` - Может выполняться параллельно
- `[Зависимость:X]` - Требует предварительного завершения задачи X
- `[P0/P1/P2]` - Маркировка приоритета

### 3. Упорядочивание и группировка задач

- Группировка по приоритетам
- Определение зависимостей
- Маркировка параллельных задач
- Оценка времени выполнения

### 4. Создание плана выполнения

```markdown
## План выполнения

### Первый этап (1-я неделя)
Параллельная группа задач 1:
- Проработка главного героя [P]
- Основы мировоззрения [P]
- Черновик главы 1 [P]

### Второй этап (2-3 недели)
Последовательные задачи:
- Глава 2 [Зависимость: Глава 1]
- Глава 3 [Зависимость: Глава 2]

Параллельная группа задач 2:
- Проработка второстепенных персонажей [P]
- Дизайн сцен [P]
```

### 5. Вывод статистики по задачам

- Общее количество задач
- Ожидаемое общее количество слов
- Ожидаемое время выполнения
- Ключевые этапы
