## Заметки по созданию

### Напоминания по «Кругу историй»
- [ ] Каждый шаг должен быть чётко различим
- [ ] Потребность должна быть достаточно сильной
- [ ] Новый мир должен быть действительно «новым»
- [ ] Процесс поиска должен иметь взлёты и падения
- [ ] Найденное не обязательно является тем, что хотелось
- [ ] Цена должна быть весомой
- [ ] Возвращение — это не просто возвращение
- [ ] Изменение должно быть реальным

### Распространённые ошибки
- Слишком поверхностная потребность
- Недостаточно чуждый новый мир
- Слишком лёгкий процесс поиска
- Недостаточно тяжёлая цена
- Недостаточно глубокое изменение

### Техники применения цикла
- Можно нарушить цикл, чтобы создать трагедию
- Можно ускорить цикл, чтобы увеличить темп
- Можно вложить циклы друг в друга для увеличения глубины
- Можно использовать обратный цикл для создания новизны
