# Запрос для Gemini по анализу романа

## Как использовать
1. Скопируйте полный запрос ниже
2. Замените [X] на фактическое количество глав
3. Вставьте содержание вашего романа после запроса
4. Gemini вернет структурированные предложения, которые можно напрямую использовать в Novel Writer

---

## Пример диалога

### Ввод пользователя
```
Проанализируй первые 10 глав моего романа и дай предложения по улучшению. Пожалуйста, выводи строго в формате JSON, указанном ниже, без каких-либо дополнительных пояснений.

[Шаблон JSON...]

Проанализируй следующее содержание романа:
Глава 1: xxx
Глава 2: xxx
...
```

### Gemini должен вернуть
```json
{
  "version": "1.0",
  "source": "Gemini",
  "analysis_date": "2025-01-24",
  "chapters_analyzed": {
    "start": 1,

  },

## Использование возвращенных предложений

1. Скопируйте полный JSON, возвращенный Gemini
2. Выполните в Claude Code:
```
/constitution refine [Вставьте JSON]
```
3. Система автоматически интегрирует предложения в ваши правила написания

## Примечания
- Убедитесь, что Gemini возвращает чистый JSON
- Если возвращенное содержимое содержит дополнительный текст, скопируйте только JSON-часть
- Рекомендуется проводить анализ каждые 10 глав для постоянного улучшения
