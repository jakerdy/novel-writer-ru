# Активация творческого режима «Каменный осьминог»

Запустите эпический творческий режим, загрузите полную систему стилей для масштабных повествований, глубоких интриг и ярких персонажей.

## Как использовать

```
/shizhangyu-style [опции]
```

## Опции

- `--mode <type>` - Выбор режима творчества:
  - `epic` - Эпическое повествование (по умолчанию)
  - `intrigue` - С акцентом на интриги
  - `character` - С акцентом на персонажей
  - `mixed` - Смешанный режим

- `--era <period>` - Исторический период:
  - `ancient` - Древняя история
  - `modern` - Современность
  - `alternate` - Альтернативный мир

- `--length <number>` - Количество слов в главе (по умолчанию 20000)

## Описание функций

После активации будет автоматически:

1. **Загружен эпический шаблон**
   - Масштабная историческая предпосылка
   - Многоуровневая система интриг
   - Многогранные персонажи

2. **Инициализирована система творчества**
   - Режим сверхдлинных глав по 20000 слов
   - Таблица отслеживания сюжетных завязок
   - Валидатор логики интриг
   - Движок персонализации речи персонажей

3. **Настроены параметры повествования**
   - Масштаб повествования: Эпический
   - Сложность интриг: Чрезвычайно высокая
   - Глубина персонажей: Многогранная и полная
   - Плотность сюжетных завязок: Высокая
   - Богатство деталей: Максимальное

4. **Включены специальные функции**
   - Генерация сильного крючка в начале
   - Управление параллельными сюжетными линиями
   - Контроль темпа кульминации
   - Разработка неожиданных концовок

## Примечания

- Этот режим особенно подходит для исторических, интриганских, деловых и военных жанров.
- По умолчанию главы выводятся по 20000 слов, пожалуйста, выделите достаточно времени.
- Таблица отслеживания сюжетных завязок поддерживается автоматически, ручное управление не требуется.
- Рекомендуется сначала продумать план, а затем приступать к творчеству.

## Продвинутые приёмы

1. **Параллельные сюжетные линии**: Можно одновременно вести 3-5 сюжетных линий.
2. **Связывание сюжетных завязок**: В каждой главе закладывайте минимум 5 сюжетных завязок.
3. **Речь персонажей**: Присвойте главным персонажам уникальные речевые обороты.
4. **Контроль темпа**: Распределите 3 точки кульминации в начале, середине и конце.

## Связанные команды

- `/shizhangyu-write` - Начать эпическое творчество
- `/shizhangyu-plot` - Разработать план интриг
- `/shizhangyu-character` - Создать персонажа
- `/shizhangyu-foreshadow` - Управлять сюжетными завязками
- `/expert shizhangyu-master` - Вызвать мастера для руководства
