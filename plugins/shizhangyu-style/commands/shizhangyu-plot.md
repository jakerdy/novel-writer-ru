# Мастер дизайна интриг в стиле Шичжанъюй

Создание многоуровневых, многосторонних и логически обоснованных интриг, создающих глубокие игры в стиле Шичжанъюй.

## Использование

```
/shizhangyu-plot <тип> [описание]
```

## Типы интриг

- `palace` - Дворцовые интриги
- `political` - Политические игры
- `military` - Военные стратегии
- `business` - Бизнес-войны
- `succession` - Борьба за наследство
- `alliance` - Союзы и предательства
- `infiltration` - Проникновение и шпионаж
- `revolution` - Революции и перевороты

## Особенности

### 1. Анализ сил
- Определение сторон и их интересов
- Анализ соотношения сил и распределения ресурсов
- Оценка преимуществ и недостатков каждой стороны
- Прогнозирование возможных союзов и предательств

### 2. Разработка уловок
- Базовая библиотека уловок (36 стратагем)
- Комбинации продвинутых стратегий
- Многоуровневый дизайн
- Введение неожиданных поворотов и сюрпризов

### 3. Проверка логики
- Проверка осуществимости уловок
- Верификация причинно-следственных связей
- Обеспечение разумности мотивов персонажей
- Избегание логических пробелов

## Пример использования

### Разработка дворцовой интриги
```
Пользователь: /shizhangyu-plot palace борьба за трон наследника

AI:
【Интрига: Битва за наследство】

Анализ сил:
━━━━━━━━━━━━━━━━━━
Лагерь 1: Партия наследника
- Лидер: Наследный принц Ли Чэнцянь
- Преимущества: Законное право на престол, поддержка дворцовой стражи, часть придворных
- Недостатки: Подозрения императора, дурная репутация, слабая военная поддержка
- Ключевые фигуры: Наставник наследника Вэй Чжэн, генерал левой гвардии Хоу Цзюньцзи

Лагерь 2: Партия принца Вэй Ли Тай
- Лидер: Принц Вэй Ли Тай
- Преимущества: Благосклонность императора, талантлив, богат
- Недостатки: Отсутствие военной власти, неприязнь придворных к его высокомерию
- Ключевые фигуры: Канцлер Цэнь Вэньбэнь, главный инспектор Лю Цзи

Лагерь 3: Партия принца Цзинь (скрытая сила)
- Лидер: Принц Цзинь Ли Чжи
- Преимущества: Скромность, поддержка императрицы, никто не подозревает
- Недостатки: Самый младший, кажется слабым
- Ключевые фигуры: Чжансунь Уцзи (тайная поддержка)

Основная линия интриги:
Первый уровень: Очевидная борьба между наследником и принцем Вэй
Второй уровень: Чжансунь Уцзи тайно ослабляет обоих сильных претендентов
Третий уровень: Император намеренно сталкивает сыновей для баланса сил
Четвертый уровень: Внешние силы (родственники императрицы) вмешиваются для долгосрочных целей

Конкретные шаги:

Этап 1: Разжигание розни (1-3 месяца)
- Партия принца Вэй фабрикует доказательства измены наследника
- Партия наследника распространяет слухи о тайных сговорах принца Вэй
- Партия принца Цзинь подстрекает обе стороны

Этап 2: Удар под дых (4-6 месяцев)
- Организовать устранение Хоу Цзюньцзи (наследник теряет военную поддержку)
- Обвинить Цэнь Вэньбэня во взяточничестве (принц Вэй теряет поддержку канцлера)
- Чжансунь Уцзи сохраняет нейтралитет, тайно помогает принцу Цзинь

Этап 3: Финальный удар (7-9 месяцев)
- План военного мятежа наследника раскрыт, он низложен
- Принц Вэй теряет бдительность, раздражает императора
- Принц Цзинь "вынужден" принять титул

Ключевые поворотные моменты:
1. Дело о мятеже Хоу Цзюньцзи (частично правда)
2. Раскрытие заговора Чэнцяня и принца Хань
3. Расчеты принца Вэй во время "уступки"
4. Ключевой голос Чжансунь Уцзи

Скрытые элементы:
- Тайное письмо трехлетней давности
- Личность предателя в резиденции наследника
- Истинные намерения императора
- Момент вмешательства У Мэйнян
```

## Продвинутые функции

### Комбинатор стратегий
Позволяет комбинировать несколько базовых стратегий в сложные планы:
- Открытое движение по главной дороге + Тайное движение по обходному пути + Осада Вэй для спасения Чжао
- Стратегия «красавицы» + Стратегия «двойного агента» + Стратегия «разделяй и властвуй»
- Стратегия «пустого города» + Стратегия «притворства» + Стратегия «ложной капитуляции»

### Симулятор логики
Моделирует возможные результаты применения стратегий:
- Оценка вероятности успеха
- Выявление рисков
- Разработка планов действий в чрезвычайных ситуациях
- Подготовка контрмер

### Анализ мотивов персонажей
Глубокий анализ мотивов каждого участника:
- Явные цели
- Истинные намерения
- Минимальные принципы
- Возможные точки предательства

## Формат вывода

1. **Карта сил**：Четкое отображение отношений между сторонами
2. **План стратегий**：Детальный план по этапам
3. **Ключевые узлы**：Поворотные моменты, определяющие успех
4. **Оценка рисков**：Возможные факторы неудачи
5. **Список зацепок**：Необходимые для внедрения подсказки
