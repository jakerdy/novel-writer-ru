# Мастер разработки интриг

Разрабатывайте многоуровневые, многосторонние и логически обоснованные интриги, создавая глубокие игры в стиле Шичжанъюй.

## Использование

```
/shizhangyu-plot <тип> [описание]
```

## Типы интриг

- `palace` - Дворцовые интриги
- `political` - Политические игры
- `military` - Военные стратегии
- `business` - Бизнес-войны
- `succession` - Борьба за наследство
- `alliance` - Союзы и предательства
- `infiltration` - Проникновение и шпионаж
- `revolution` - Революции и перевороты

## Функциональные особенности

### 1. Анализ сил
- Определение сторон и их интересов
- Анализ соотношения сил и распределения ресурсов
- Оценка преимуществ и недостатков каждой стороны
- Прогнозирование возможных союзов и предательств

### 2. Разработка замыслов
- Базовая библиотека замыслов (36 стратагем)
- Комбинации продвинутых стратегий
- Многоуровневое вложенное проектирование
- Установка поворотов и неожиданностей

### 3. Проверка логики
- Проверка осуществимости замыслов
- Проверка причинно-следственных связей
- Обеспечение разумности мотивации персонажей
- Предотвращение логических пробелов

## Пример использования

### Разработка дворцовой интриги
```
Пользователь: /shizhangyu-plot palace Борьба за трон наследника

AI:
【Интрига: Битва за наследство】

Анализ сил:
━━━━━━━━━━━━━━━━━━
Фракция 1: Партия наследника
- Лидер: Наследный принц Ли Чэнцянь
- Преимущества: Законный титул, система Восточного дворца, поддержка части придворных
- Недостатки: Подозрительность императора, дурное поведение, слабая военная мощь
- Ключевые фигуры: Наставник наследника Вэй Чжэн, генерал левой гвардии Хоу Цзюньцзи

Фракция 2: Партия принца Вэй
- Лидер: Принц Вэй Ли Тай
- Преимущества: Благосклонность императора, выдающиеся таланты, богатые ресурсы
- Недостатки: Отсутствие военной власти, неприязнь придворных к его высокомерию
- Ключевые фигуры: Канцлер Цэнь Вэньбэнь, цензор императорской канцелярии Лю Цзи

Фракция 3: Партия принца Цзинь (скрытая сила)
- Лидер: Принц Цзинь Ли Чжи
- Преимущества: Сдержанность и терпение, поддержка императрицы, отсутствие бдительности со стороны других
- Недостатки: Самый молодой, кажется слабым
- Ключевые фигуры: Чжансунь Уцзи (тайная поддержка)

Основная линия интриги:
Первый уровень: Очевидное соперничество между наследником и принцем Вэй
Второй уровень: Чжансунь Уцзи тайно ослабляет обоих сильных претендентов
Третий уровень: Император намеренно сталкивает сыновей, чтобы сохранить баланс
Четвертый уровень: Влияние внешних сил, стремящихся к долгосрочным выгодам

Конкретные шаги:

Этап 1: Разжигание розни (1-3 месяца)
- Партия принца Вэй фабрикует доказательства заговора наследника
- Партия наследника распространяет слухи о тайных сговорах принца Вэй
- Партия принца Цзинь подливает масла в огонь с обеих сторон

Этап 2: Подрыв основ (4-6 месяцев)
- Организовать устранение Хоу Цзюньцзи (наследник теряет военную поддержку)
- Обвинить Цэнь Вэньбэня во взяточничестве (принц Вэй теряет поддержку центральной власти)
- Чжансунь Уцзи сохраняет нейтралитет, тайно помогая принцу Цзинь

Этап 3: Финальный удар (7-9 месяцев)
- Провал заговора наследника, его низложение
- Принц Вэй теряет бдительность, вызывая гнев императора
- Принц Цзинь «вынужден» принять титул

Ключевые поворотные моменты:
1. Дело о мятеже Хоу Цзюньцзи (частично правда)
2. Раскрытие заговора Чэнцяня и принца Хань
3. Расчетливое «смирение» принца Вэй в нужный момент
4. Решающий голос Чжансунь Уцзи

Установка зацепок:
- Тайное письмо трехлетней давности
- Личность внутреннего предателя в резиденции наследника
- Истинные намерения императора
- Момент вмешательства У Мэйнян
```

## Расширенные функции

### Комбинатор стратегий
Позволяет объединять несколько базовых стратегий в сложные:
- Открытое укрепление дорог + Тайное пересечение рек + Осада Вэй для спасения Чжао
- Стратегия обольщения + Стратегия контрразведки + Стратегия внесения раздора
- Стратегия пустого города + Стратегия самоистязания + Стратегия ложной капитуляции

### Симулятор логики
Моделирует возможные результаты применения стратегий:
- Оценка вероятности успеха
- Выявление рисков
- Разработка планов действий в чрезвычайных ситуациях
- Подготовка контрмер

### Анализ мотивации персонажей
Глубокий анализ мотивов каждого участника:
- Очевидные цели
- Истинные намерения
- Нижние границы принципов
- Возможные точки предательства

## Формат вывода

1. **Карта сил**: Четкое отображение взаимоотношений сторон
2. **План стратегии**: Детальный план по этапам
3. **Ключевые точки**: Поворотные моменты, определяющие успех
4. **Оценка рисков**: Возможные факторы неудачи
5. **Список зацепок**: Необходимые для внедрения подсказки

## Связанные команды

- `/shizhangyu-style` - Активация стиля
- `/shizhangyu-write` - Реализация написания
- `/shizhangyu-character` - Согласование с персонажами
- `/shizhangyu-foreshadow` - Согласование зацепок
