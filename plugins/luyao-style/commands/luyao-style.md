# Активация режима творчества в стиле Лу Яо

Запуск режима реалистичного литературного творчества, загрузка простого стиля «Обыкновенного мира» и философских размышлений о жизни.

## Способ применения

```
/luyao-style [опции]
```

## Опции

- `--era <период>` - Исторический фон:
  - `1970s` - Семидесятые (период производственной бригады)
  - `1980s` - Восьмидесятые (начало политики реформ и открытости) по умолчанию
  - `1990s` - Девяностые (период рыночной экономики)

- `--setting <тип>` - Настройка сцены:
  - `rural` - Сельская местность (по умолчанию)
  - `urban` - Город
  - `mining` - Горнодобывающий район
  - `factory` - Фабрика

- `--length <число>` - Количество символов в главе (по умолчанию 3000)

## Описание функций

После активации будет автоматически выполнено:

1. **Загрузка шаблона реализма**
   - Простой и глубокий стиль повествования
   - Эпическая история обычных людей
   - Грандиозный фон эпохи перемен

2. **Инициализация системы жизни**
   - Библиотека описания бытовых деталей
   - Система лексики северо-шаньсийского диалекта
   - База данных исторических фонов эпохи
   - Шаблон народных обычаев и нравов

3. **Настройка параметров творчества**
   - Степень реализма: чрезвычайно высокая
   - Степень сдержанности эмоций: сдержанная и глубокая
   - Плотность деталей: богатая и правдивая
   - Использование диалектов: умеренное включение
   - Индекс надежды: свет в страданиях

4. **Включение специальных функций**
   - Автоматическое включение бытовых деталей
   - Внедрение отпечатков эпохи
   - Отслеживание судеб персонажей
   - Контроль эмоционального ритма

## Примечания

- Этот режим особенно подходит для тем реализма и деревенской литературы.
- Акцент делается на правдивости, все детали должны соответствовать логике эпохи.
- Обратите внимание на сдержанное выражение эмоций, избегайте прямолинейной сентиментальности.
- Всегда сохраняйте глубокую заботу об обычных людях.

## Продвинутые приемы

1. **Внедрение эпохи**: Сделайте исторический фон частью истории.
2. **Правдивость деталей**: Создайте правдоподобный мир с помощью бытовых деталей.
3. **Сдержанность эмоций**: Выражайте самые глубокие эмоции самым простым способом.
4. **Тема надежды**: Даже в самых трудных обстоятельствах должна быть искорка света.

## Связанные команды

- `/luyao-write` - Начать реалистичное творчество
- `/luyao-character` - Создание обычных персонажей
- `/luyao-detail` - Генерация бытовых деталей
- `/luyao-dialogue` - Создание диалогов в народном стиле
- `/expert luyao-master` - Вызвать мастера Лу Яо
