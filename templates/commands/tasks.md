Создание конкретного, выполнимого списка задач на основе плана творчества.

## Цель

Преобразование макроплана в микрозадачи, делая творческий процесс управляемым и отслеживаемым.

## Шаги выполнения

### 1. Загрузка документов плана

Запустите `{SCRIPT}` для загрузки:
- План творчества: `stories/*/creative-plan.md`
- Информация об архитектуре глав
- Временная шкала и зависимости

### 2. Генерация списка задач

Создайте `stories/*/tasks.md`, содержащий:

#### Основные задачи написания

**Важно**: На основе главы 5 из specification.md и распределения сюжетных линий из creative-plan.md, пометьте каждую задачу написания, какие сюжетные линии она затрагивает.

```markdown
## Задачи написания

### Высокий приоритет [Должно быть выполнено в первую очередь]

- [ ] [P0] **T001** - Глава 1: [Название главы] (Целевое количество слов)
  - **Основная задача**: [Основная задача этой главы]
  - **Ключевые сюжетные моменты**: [Конкретные сюжетные моменты]
  - **Задействованные сюжетные линии**:
    - PL-XX([Название сюжетной линии]) ⭐⭐⭐ Основное продвижение
    - PL-YY([Название сюжетной линии]) ⭐ Фоновое
  - **Точки пересечения**: [Если применимо] X-001([Описание точки пересечения])
  - **Завязка/Развязка**: [Если применимо] Завязка F-001 / Развязка F-002
  - **Должно включать**: [Список ключевых элементов]
  - **Крючок в конце главы**: [Создание интриги]
  - **Зависимости**: Нет
  - **Выход**: `content/volume1/chapter-001.md`

- [ ] [P0] **T002** - Профили персонажей: Подробное описание главного героя
  - **Основная задача**: Доработка описания главного героя
  - **Должно включать**: Характер, предыстория, способности, желания, страхи, дуга развития
  - **Зависимости**: Нет
  - **Выход**: `characters/protagonist.md`

### Средний приоритет [Нормальное продвижение]

- [ ] [P1] **T005** - Глава 5: [Название главы] (Целевое количество слов)
  - **Основная задача**: [Основная задача этой главы]
  - **Ключевые сюжетные моменты**: [Конкретный сюжет]
  - **Задействованные сюжетные линии**:
    - PL-01([Название сюжетной линии]) ⭐⭐⭐ Основное продвижение
    - PL-02([Название сюжетной линии]) ⭐⭐ Вспомогательное
  - **Точки пересечения**: Нет
  - **Завязка/Развязка**: Завязка F-001([Содержание завязки])
  - **Должно включать**: [Ключевые элементы]
  - **Крючок в конце главы**: [Интрига]
  - **Зависимости**: T004 (Глава 4)
  - **Выход**: `content/volume1/chapter-005.md`

### Низкий приоритет [Необязательное улучшение]

- [ ] [P2] Дополнительная история: Предыстория персонажа
- [ ] [P2] Сборник материалов: Подробное описание мира
```

**Пояснение по полям задачи**:
- **Задействованные сюжетные линии**: Читать из столбца "Активные сюжетные линии" в creative-plan.md, указать, какие сюжетные линии продвигаются в этой главе
- **Точки пересечения**: Читать из раздела 5.3 specification.md, указать, является ли эта глава точкой пересечения
- **Завязка/Развязка**: Читать из раздела 5.4 specification.md, указать, какие операции с завязками выполняются в этой главе

#### Пояснение к маркировке задач
- `[P]` - Может выполняться параллельно
- `[Зависимость:X]` - Требует предварительного завершения задачи X
- `[P0/P1/P2]` - Маркировка приоритета

### 3. Сортировка и группировка задач

- Группировка по приоритету
- Определение зависимостей
- Маркировка параллельных задач
- Оценка времени выполнения
