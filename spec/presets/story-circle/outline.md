## Вложенность циклов глав

### Большой цикл (вся книга)
Вся книга представляет собой полный 8-шаговый цикл.

### Средний цикл (арки)
Каждая арка (примерно 20 глав) может содержать небольшой цикл.

### Малый цикл (глава)
Отдельная глава также может иметь мини-цикл:
1. Начало покоя
2. Возникновение потребности
3. Предпринятие действий
4. Поиск ответов
5. Получение результата
6. Уплата цены
7. Возвращение к покою
8. Оставленные изменения

## Напоминания по написанию

### Ключевые моменты написания для каждого шага
1. **Ты**: Вызвать у читателя сопереживание текущей ситуации.
2. **Потребность**: Сделать потребность неодолимой.
3. **Вход**: Поразить читателя новым миром.
4. **Поиск**: Сделать процесс полным перипетий.
5. **Обретение**: Сделать открытие неожиданным.
6. **Цена**: Сделать потерю незабываемой.
7. **Возвращение**: Сделать возвращение значимым.
8. **Изменение**: Сделать изменения правдоподобными.

### Целостность цикла
- Убедиться, что все 8 шагов четко различимы.
- Каждый шаг должен способствовать внутренним изменениям.
- Сохранять баланс цикла.
- Концовка должна перекликаться с началом.
