```markdown
# Активация режима творчества в стиле Ван Юй

Настройте и активируйте режим творчества в стиле Ван Юй (автор «Путь бессмертного» - Fanren Xiuxian Zhuan), загрузив полные параметры стиля и систему мировоззрения.

## Как использовать

```
/wangyu-style [опции]
```

## Опции

- `--intensity <1-10>` - Интенсивность стиля (по умолчанию: 8)
- `--focus <aspect>` - Усиление определённого аспекта:
  - `world` - Построение мировоззрения
  - `character` - Создание персонажей
  - `suspense` - Создание интриги
  - `resource` - Система ресурсов
  - `battle` - Описание сражений

## Описание функций

После активации будет автоматически выполнено:

1. **Загрузка шаблона стиля**
   - Спокойный и объективный стиль повествования
   - Строгое мировоззрение мира бессмертных
   - Сложная сеть взаимоотношений персонажей

2. **Инициализация настроек мира**
   - Система восьми уровней совершенствования
   - Четыре типа систем ресурсов для совершенствования
   - Многоуровневая структура сил

3. **Настройка параметров творчества**
   - Дистанция повествования: отстранённая объективность
   - Эмоциональная интенсивность: сдержанная
   - Плотность описаний: подробная и точная
   - Уровень интриги: постепенное нарастание

4. **Включение специальных функций**
   - Система отслеживания зацепок
   - Проверка дефицита ресурсов
   - Проверка системы силы
   - Аудит логики поведения персонажей

## Примеры

```

✓ Система мировоззрения мира совершенствования (от «Заложения основ» до «Великого слияния» — восемь уровней)
✓ Система дефицита ресурсов (духовные камни, пилюли, магические артефакты, техники совершенствования)
✓ Спокойный и объективный стиль повествования
✓ Шаблон осторожного и прагматичного главного героя
✓ Трекер интриг и зацепок

Включены характеристики стиля:
- Повествование сохраняет спокойствие и отстранённость
- Подчёркивается трудность получения ресурсов
- Отношения между персонажами переплетены интересами
- Сражения сосредоточены на стратегии и деталях
- Автоматически закладываются долгосрочные зацепки

Вы можете начать творить! Рекомендуется использовать команду /wangyu-write для начала письма.

```
## Примечания

- Этот режим особенно подходит для жанров сянься и фэнтези.
- Рекомендуется использовать команду `/track` для управления зацепками.
- Может сочетаться с другими методами письма (например, «Путешествие героя»).
- Активация влияет на все последующие команды письма.

## Связанные команды

- `/wangyu-write` - Писать в стиле Ван Юй
- `/wangyu-analyze` - Анализировать соответствие стилю
- `/wangyu-enhance` - Усилить характеристики стиля Ван Юй
- `/expert wangyu-master` - Вызвать мастера стиля Ван Юй
```
