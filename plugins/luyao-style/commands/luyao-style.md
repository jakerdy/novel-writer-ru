# Активация режима творчества в стиле Лу Яо

Запуск режима создания реалистической литературы, загрузка простоты «обычного мира» и философских размышлений о жизни.

## Использование

```
/luyao-style [опции]
```

## Опции

- `--era <период>` - Исторический период:
  - `1970s` - 1970-е (период производственных бригад)
  - `1980s` - 1980-е (начало политики реформ и открытости) по умолчанию
  - `1990s` - 1990-е (период рыночной экономики)

- `--setting <тип>` - Настройка места действия:
  - `rural` - Сельская местность (по умолчанию)
  - `urban` - Город
  - `mining` - Шахтёрский район
  - `factory` - Завод

- `--length <число>` - Количество слов в главе (по умолчанию 3000)

## Описание функций

После активации автоматически будет выполнено:

1. **Загрузка шаблона реализма**
   - Простой и глубокий стиль повествования
   - Эпическая сага о жизни обычных людей
   - Масштабный фон эпохи и её перемен

2. **Инициализация системы жизнеописания**
   - База данных описаний бытовых деталей
   - Лексическая система диалектов северо-западной провинции Шэньси
   - База данных исторических сведений эпохи
   - Шаблон народных обычаев и нравов

3. **Конфигурация параметров творчества**
   - Степень реализма: чрезвычайно высокая
   - Степень сдержанности эмоций: сдержанная и глубокая
   - Плотность деталей: богатая и достоверная
   - Использование диалектов: умеренное включение
   - Индекс надежды: свет в конце тоннеля сквозь страдания

4. **Включение специальных функций**
   - Автоматическое включение бытовых деталей
   - Внедрение отпечатков эпохи
   - Отслеживание судеб персонажей
   - Контроль эмоционального ритма

✓ Лексическая система диалектов северо-западной провинции Шэньси

Включены стилистические особенности:
- Простой и незамысловатый язык

Готово! Вы можете начать рассказывать эти простые, но великие истории.

## Примечания

- Этот режим особенно подходит для реалистических и деревенских тем.
- Акцент делается на достоверности, все детали должны соответствовать логике эпохи.
- Обратите внимание на сдержанное выражение эмоций, избегайте прямолинейной сентиментальности.
- Всегда сохраняйте глубокое сочувствие к простым людям.

## Продвинутые приемы

1. **Внедрение эпохи**: Сделайте исторический фон частью истории.
2. **Достоверность деталей**: Создайте правдоподобный мир с помощью бытовых деталей.
3. **Сдержанность эмоций**: Выражайте самые глубокие чувства самым простым способом.
4. **Тема надежды**: Даже в самой трудной ситуации должна быть искорка света.

## Связанные команды

- `/luyao-write` - Начать реалистическое творчество
- `/luyao-character` - Создать образ обычного человека
- `/luyao-detail` - Сгенерировать бытовые детали
- `/luyao-dialogue` - Создать диалог в народном стиле
- `/expert luyao-master` - Вызвать мастера Лу Яо
