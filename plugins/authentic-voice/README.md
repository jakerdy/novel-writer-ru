# Плагин Authentic Voice

Сделайте текст похожим на ваш собственный: подчеркните детали из жизни, разговорный ритм и последовательность индивидуального словарного запаса. Этот плагин фокусируется на «повышении оригинальности и естественности», он не связан и не используется для каких-либо сценариев обнаружения и обхода.

## Установка

```
novel plugins add authentic-voice
```

## Подготовка личных материалов

Заполните в `.specify/memory/personal-voice.md`:
- Коронные фразы и часто используемые выражения
- Фиксированные стили предложений и предпочтения в ритме
- Список жизненного опыта и отраслевая/региональная лексика
- Писательские табу и избегаемые темы
- Предпочтения в метафорах и банк образов

## Часто используемые команды

- `/authentic-voice` Запуск написания с использованием аутентичного голоса, автоматическое чтение личных материалов и предоставление «карточки с источниками» перед выводом.
- `/authenticity-audit` Проведение самопроверки на «человечность» существующего текста и предложение по его переработке.
- `/expert authentic-editor` Вызов эксперта по аутентичному редактированию для более детальной переработки на уровне строк.

## Применимые сценарии

- Создание основного текста: требуется стабильный личный стиль и ритм.
- Вторичная доработка: избегание пустых фраз, шаблонности и однообразного ритма.
- Долгосрочные проекты: постепенное накопление личного словарного запаса и образов для формирования стабильного «авторского отпечатка».
