# Активация режима творчества в стиле Ван Юй

Настройте и активируйте режим творчества в стиле Ван Юй (автор «Пути бессмертного», также известного как «Путь самосовершенствования смертного») с полной загрузкой параметров стиля и системы мировоззрения.

## Использование

```
/wangyu-style [опции]
```

## Опции

- `--intensity <1-10>` - Интенсивность стиля (по умолчанию: 8)
- `--focus <аспект>` - Фокусировка на определённом аспекте:
  - `world` - Построение мировоззрения
  - `character` - Формирование персонажей
  - `suspense` - Создание интриги
  - `resource` - Система ресурсов
  - `battle` - Описание сражений

## Описание функций

После активации будет автоматически выполнено:

1. **Загрузка шаблона стиля**
   - Спокойный, объективный стиль повествования
   - Строгое мировоззрение мира совершенствования
   - Сложная сеть взаимоотношений персонажей

2. **Инициализация настроек мира**
   - Восемь основных уровней совершенствования
   - Четыре типа систем ресурсов для совершенствования
   - Многоуровневая структура сил

3. **Конфигурация параметров творчества**
   - Дистанция повествования: отстранённая, объективная
   - Эмоциональная интенсивность: сдержанная, невыразительная
   - Плотность описаний: подробная, точная
   - Уровень интриги: постепенное нарастание

4. **Включение специальных функций**
   - Система отслеживания сюжетных зацепок
   - Проверка дефицита ресурсов
   - Верификация системы силы
   - Аудит логики поведения персонажей

## Примечания

- Этот режим особенно подходит для жанров сянься и фэнтези.
- Рекомендуется использовать совместно с командой `/track` для управления сюжетными зацепками.
- Может быть объединён с другими методами письма (например, «Путь героя»).
- После активации влияет на все последующие команды письма.

## Связанные команды

- `/wangyu-write` - Писать в стиле Ван Юй
- `/wangyu-analyze` - Анализировать соответствие стилю
- `/wangyu-enhance` - Усилить характеристики стиля Ван Юй
- `/expert wangyu-master` - Вызвать мастера стиля Ван Юй
