# Активация режима творчества «Стиль Шичжанъюй»

Запуск эпического режима творчества, загрузка полной системы стилей для грандиозного повествования, глубоких интриг и ярких персонажей.

## Использование

```
/shizhangyu-style [опции]
```

## Опции

- `--mode <type>` - Выбор режима творчества:
  - `epic` - Эпическое повествование (по умолчанию)
  - `intrigue` - С акцентом на интриги
  - `character` - С акцентом на персонажей
  - `mixed` - Смешанный режим

- `--era <period>` - Исторический период:
  - `ancient` - Древняя история
  - `modern` - Современность
  - `alternate` - Альтернативный мир

- `--length <number>` - Количество символов в главе (по умолчанию 20000)

## Описание функций

После активации будет автоматически выполнено:

1. **Загрузка эпического шаблона**
   - Грандиозная фоновая установка эпохи
   - Многоуровневая система интриг
   - Объемный ансамбль персонажей

2. **Инициализация системы творчества**
   - Режим сверхдлинных глав по 20000 символов
   - Таблица отслеживания сюжетных завязок
   - Валидатор логики интриг
   - Движок для персонализации речи персонажей

3. **Настройка параметров повествования**
   - Масштаб повествования: Эпический
   - Сложность интриг: Чрезвычайно высокая
   - Глубина персонажей: Объемные и многогранные
   - Плотность сюжетных завязок: Высокая
   - Богатство деталей: Максимальное

4. **Включение специальных функций**
   - Генерация сильного крючка в начале
   - Управление параллельными сюжетными линиями
   - Контроль ритма кульминации
   - Разработка неожиданных финалов

## Примечания

- Данный режим особенно подходит для исторических, политических, деловых и военных тем.
- По умолчанию выводятся главы по 20000 символов, пожалуйста, заложите достаточно времени.
- Таблица отслеживания сюжетных завязок поддерживается автоматически, ручное управление не требуется.
- Рекомендуется сначала продумать план, а затем приступать к творчеству.

## Продвинутые приемы

1. **Параллельные линии**：Можно одновременно вести 3-5 сюжетных линий.
2. **Связывание сюжетных завязок**：В каждой главе закладывайте как минимум 5 сюжетных завязок.
3. **Речь персонажей**：Присвойте основным персонажам уникальные речевые обороты.
4. **Контроль ритма**：3 точки кульминации распределите в начале, середине и конце.
