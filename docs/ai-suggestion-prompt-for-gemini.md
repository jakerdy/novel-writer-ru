# Промпт для Gemini для анализа романа

## Как использовать
1. Скопируйте полный промпт ниже
2. Замените [X] на фактическое количество глав
3. Вставьте содержание вашего романа после промпта
4. Gemini вернет структурированные рекомендации, которые можно напрямую использовать в Novel Writer

---

## Использование возвращенных рекомендаций

1. Скопируйте полный JSON, возвращенный Gemini
2. Выполните в Claude Code:
```
/constitution refine [вставьте JSON]
```
3. Система автоматически интегрирует рекомендации в ваши правила создания контента

## Примечания
- Убедитесь, что Gemini возвращает чистый JSON-формат
- Если возвращенное содержимое содержит дополнительный текст, скопируйте только JSON-часть
- Рекомендуется проводить анализ каждые 10 глав для постоянного улучшения
