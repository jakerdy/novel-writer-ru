2. **达 (Беглость)**
   - Плавность языка: Естественное английское выражение.
   - Соответствующий ритм: Соответствие привычкам чтения на английском языке.

3. **雅 (Элегантность)**

   - Передача эмоций: Точная передача эмоционального напряжения.

- Core Formation → Golden Core
- Nascent Soul → Spirit Embryo
- Tribulation → Heavenly Trial
- Создание единой системы терминологии.

### 2. Трансформация системы обращений
**Трудность**: Шифу (Учитель), Шиcюн (Старший брат), Даою (Товарищ по дао) и т.д.

- 师傅 → Master/Teacher
- 师兄 → Senior Brother
