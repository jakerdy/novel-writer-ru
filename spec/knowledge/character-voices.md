## Запись эволюции языка

### Рост языка главного героя
1. **Начальный этап** (главы 1-20)
   - Характеристики: Неопытный, неуверенный
   - Пример: "Я... я не совсем уверен..."

2. **Средний этап** (главы 21-60)
   - Характеристики: Постепенно взрослеет, обретает уверенность
   - Пример: "Я считаю, что следует поступить так."

3. **Поздний этап** (после главы 61)
   - Характеристики: Зрелый, авторитетный
   - Пример: "Слушайте мой приказ, немедленно действуйте!"

## Языковые табу

### Чего следует избегать
- **Смешение современных выражений с историческим сеттингом**
  - Неправильный пример: "ОК", "Пока", "Вау"
  - Правильный пример: "Хорошо", "Прощайте", "Боже мой"

- **Язык, не соответствующий характеру**
  - Персонажи с утонченным характером должны избегать грубости
  - Грубые персонажи должны избегать изысканной речи
  - Дети должны избегать взрослой лексики

- **Несоответствие географии**
  - Жители севера не должны использовать южные диалекты
  - Персонажи из древности не должны использовать иностранные слова
