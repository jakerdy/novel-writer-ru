# Система управления предзнаменованиями

Мастер раскладки предзнаменований в стиле Шичжанъюй, управляющий тысячами скрытых нитей и создающий неожиданные концовки.

## Как использовать

```
/shizhangyu-foreshadow <действие> [параметры]
```

## Типы действий

- `add` - добавить новое предзнаменование
- `list` - просмотреть список предзнаменований
- `track` - отслеживать статус предзнаменования
- `activate` - активировать предзнаменование
- `resolve` - разрешить предзнаменование
- `analyze` - анализировать раскладку предзнаменований
- `suggest` - получить предложения по предзнаменованиям
- `check` - проверить логическую цепочку

## Типы предзнаменований

### Предзнаменования-объекты
- Письма, секретные послания, жетоны, карты
- Предметы, кажущиеся обычными, но на самом деле ключевые
- Судьба утерянных предметов
- Источник таинственных подарков

### Предзнаменования-диалоги
- Случайно раскрытая информация
- Пророчества и предсказания
- Каламбуры и намеки
- Шутки, содержащие правду

### Предзнаменования-события
- Встречи, кажущиеся совпадениями
- Мелкие, но необычные события
- События, происходящие на фоне
- Проигнорированные детали

### Предзнаменования-персонажи
- Таинственное происхождение
- Скрытая личность
- Невысказанные секреты
- Пропавшие ключевые персонажи

3. Может быть ключом к открытию тайны
4. Может быть связан с происхождением главного героя

- Скоро (главы 5-10): как удостоверение личности
- В середине (главы 20-30): как ключ к важной разгадке

Связанные предзнаменования:

## Расширенные функции

### Дизайн цепочек предзнаменований
Создание каскадных реакций из нескольких предзнаменований:
- Основная цепочка предзнаменований (ключевая тайна)
- Побочные цепочки предзнаменований (дополнительная информация)
- Преднамеренные предзнаменования (создание интриги)
- Предзнаменования-перевертыши (опровержение ожиданий)

### Управление временной шкалой
Управление предзнаменованиями в хронологическом порядке:
- Немедленные предзнаменования (в течение 1-3 глав)
- Короткие предзнаменования (5-10 глав)
- Средние предзнаменования (15-30 глав)
- Длинные предзнаменования (пронизывающие всю книгу)

### Валидатор логики
Проверка логической обоснованности предзнаменований:
- Обоснована ли причинно-следственная связь
- Рационален ли временной порядок
- Адекватна ли информационная нагрузка
- Соответствует ли время разрешения

## Спецификации вывода

1. **Система нумерации**: Единая нумерация для удобства отслеживания.
2. **Маркировка статуса**: Ожидает заложения / Заложено / Активировано / Разрешено.
3. **Уровень приоритета**: Высокий / Средний / Низкий.
4. **Указание связей**: Обозначение связанных предзнаменований.
5. **Напоминание о разрешении**: Рекомендации по оптимальному времени.

## Связанные команды

- `/shizhangyu-style` - активировать стиль
- `/shizhangyu-write` - написать предзнаменование
- `/shizhangyu-plot` - предзнаменования и интриги
- `/shizhangyu-character` - предзнаменования персонажей
